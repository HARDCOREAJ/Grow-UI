# Atributes

参数 说明 类型 可选值 默认值
icon 图标的名称 String -- --
loading 加载中 Bollean -- false
iconPosition 图标的位置 String left/right left

使用小技巧

正常情况下，在加载过程中的图标是与初始时不一样的。所以在这个button组件，通过判断pros去决定在加载过程中，原始图标是否显示。源码中具体实现如下：

 <g-icon class="icon" v-if="icon&&!loading" :name="icon"></g-icon>
